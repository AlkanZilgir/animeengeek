⟨ANIMEENGEEK⟩
An e-commerce web app for anime accessories — built with plain HTML, CSS, and JavaScript. No frameworks, no build tools, just clean front-end code.

🔗 Live site: alkanzilgir.github.io/animeengeek

Features
Feature	Details
Product Catalogue	12 products with series filtering, search, sort and price range slider
Shopping Cart	Add/remove items, adjust quantities, order summary
Wishlist	Save favourite items across sessions via localStorage
Promo Codes	Built-in discount code system (ANIME20, GEEK10)
Order Checkout	Guest checkout flow with order history
User Auth	Register, login, and profile via localStorage
Gallery	Filterable photo gallery with lightbox
Responsive	Works on mobile, tablet and desktop
Dark UI	Custom dark theme with animated hero section
Pages
index.html       — Homepage with hero, featured products, collections
shop.html        — Full product catalogue with filters and pagination
cart.html        — Shopping cart and checkout
favorites.html   — Saved wishlist items
galery.html      — Gallery with tabs and lightbox
login.html       — Sign in
register.html    — Create account
Stack
HTML5 / CSS3 — custom design, no CSS framework
Bootstrap 5 — grid and utility classes
JavaScript (ES6) — cart, auth, filtering, localStorage
Bootstrap Icons — icon set
Run locally
No build step needed — just open index.html in a browser, or use a local server:

npx http-server .
# or
python -m http.server
